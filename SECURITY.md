# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MindTrace AI, please report it responsibly.

- Do NOT disclose vulnerabilities publicly
- Email details to the project maintainer
- Include steps to reproduce the issue

We will review and address valid reports promptly.

---

## Data Protection

- No sensitive personal data is stored in plaintext
- Authentication credentials are securely hashed
- ML data is anonymized before processing

---

## Responsible AI Commitment

MindTrace AI does not provide medical diagnoses.
All insights are preventive and informational only.
